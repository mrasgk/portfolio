## 💡 What & Why?

This project is absolutely not just a static website; it is an **interactive digital product**. 

As a versatile developer spanning the entire web and mobile ecosystem (Next.js, Angular, React Native, .NET 8, SQL Server), I needed a portfolio that wasn't just a basic list of links. It needed to *prove* my exact expertise in real-time. 

**Why did I build this?**
1. **Showcase Modern Frontend Mastery:** To demonstrate complex structural layouts like pure CSS Glassmorphism, 3D floating object transformations, and raw grid manipulation without relying on generic component libraries like Tailwind or Bootstrap.
2. **Tactile User Experience:** Using **Framer Motion**, I engineered physics-based recoil animations so that every button click, page scroll, and card hover feels incredibly snappy and native.
3. **Bulletproof Mobile Responsiveness:** Navigating this on an iPhone feels like using a mobile app, complete with a beautifully engineered full-screen blur overlay menu.
4. **Extreme Professional Clarity:** To boldly state my availability (**Auto-Entrepreneur in Casablanca & Global Remote**) and showcase a clear, unapologetic roadmap of my work logic and rates.

---

## 🎨 Design & Architecture 

Every pixel in this application was handcrafted. No generic app templates. No bloat.

- 🌌 **Immersive Space-Glass UI:** The design utilizes deep space dark themes, neon accents, and `backdrop-filter` rendering to create a layered "floating" visual aesthetic.
- 🎯 **3D Orbital Hero Layout:** The landing page features a precisely calculated CSS orbital ring animation with floating hexagonal skill badges rotating asynchronously.
- ⚡ **Performance Native:** By utilizing Next.js's App Router and raw CSS Modules, the site payload is exceptionally light and aggressively fast.
- 📱 **Adaptive Media Stacking:** Complex horizontal components (like the custom Split-Timeline) mathematically stack and collapse into readable vertical feeds entirely through strictly coded CSS media queries.

## 🚀 Spin It Up Locally

Want to see the raw code running directly on your machine? It's literally 4 commands.

```bash
# 1. Clone the repository down
git clone https://github.com/mrasgk/portfolio.git

# 2. Rip into the folder
cd portfolio

# 3. Install packages (Node.js required)
npm install

# 4. Launch the rocket 
npm run dev
```

Point your browser to `http://localhost:3000` and enjoy the flawlessly smooth scrolling.
